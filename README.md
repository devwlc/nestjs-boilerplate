# NestJS Boilerplate — Clean Architecture-Based

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

[![License](https://img.shields.io/github/license/devwlc/nestjs-boilerplate.svg)](https://github.com/devwlc/nestjs-boilerplate/blob/main/LICENSE)

⚠️ This project is under active development.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
